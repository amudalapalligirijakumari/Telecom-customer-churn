PROJECT NAME:'TELECOM CUSTOMER CHURN'

PROJECT SCOPE: The scope of this project is to predict customer churn in a telecom company using Machine Learning. The project uses customer account details, call usage information, service plans, and customer service interactions to determine whether a customer is likely to leave the service.

The project includes data loading, data preprocessing, conversion of categorical values into numerical values, visualization of customer churn distribution, training a Logistic Regression model, and evaluating the model using accuracy metrics.

The developed model can help telecom companies identify customers at risk of churn and support customer retention strategies. The project is based on historical telecom customer data and focuses on binary classification of churn and non-churn customers.
Conclusion:
The Telecom Customer Churn Prediction model helps identify potential churn customers using machine learning and supports better customer retention strategies.
